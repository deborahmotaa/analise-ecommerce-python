# Análise de Dados - E-commerce Brasileiro (Olist)

Este projeto utiliza **Python** e as bibliotecas **Pandas, Seaborn e Matplotlib** para extrair insights de uma base de dados real de e-commerce.

## 📊 O que foi analisado:
- Evolução do faturamento mensal.
- Identificação dos principais estados compradores.
- Análise de performance logística (prazos de entrega por UF).

## 🛠️ Tecnologias Utilizadas:
- Python 3.x
- Pandas para manipulação de dados.
- Seaborn e Matplotlib para visualização.
- VS Code com Jupyter Notebook.
